OurChat 是一个可以在 Linux，Windows 和 Macos 上运行的聊天软件。它支持所有的平台.

服务端使用 rust。得益于 rust，它有较高的性能。

该项目正处在高速开发中，并且有大量的工作要做。 截至目前，它仍然不能被直接使用。
