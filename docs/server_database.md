# 服务端的数据库设计

## 用户表
### user

| 列名     | 类型       | 用处                    |
|:-------|:---------|:----------------------|
| id     | int      | 主键，唯一的标识符             |
| ocid   | char(20) | 显示给用户的唯一id号           |
| passwd | char(30) | 用户的密码，经过sha256加密，不可还原 |
| name   | char(15) | 昵称                    |

## 好友表
### friend
| 列名        | 类型       | 用处         |
|:----------|:---------|:-----------|
| user_id   | int      | 用户         |
| friend_id | int      | 用户的好友      |
| name      | char(15) | 用户给好友起的备注名 |

## 聊天表
### chat
| 列名         | 类型       | 用处              |
|:-----------|:---------|:----------------|
| group_id   | int      | 群聊              |
| user_id    | int      | 标记用户在该群中        |
| name       | char(15) | 标记用户在群里的昵称      |
| group_name | char(30) | 标记用户对群聊的备注，默认为空 |

## 群聊表
### groupchat
| 列名         | 类型       | 用处   |
|:-----------|:---------|:-----|
| group_id   | int      | 群聊id |
| group_name | char(30) | 群聊名称 |
