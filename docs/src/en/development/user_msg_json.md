## Table of Contents

## Text Message

```json
{
  "type": 0,
  "text": "Text message content"
}
```

| Key  | ValueType | Comment                     |
| :--- | :-------- | :-------------------------- |
| type | Number    | Message type                |
| text | String    | Content of the text message |
