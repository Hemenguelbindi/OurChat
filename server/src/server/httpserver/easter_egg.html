很开心有人能找到这个页面，如果是开发者以外的人找到了这里，那么一定是 OurChat 已经被我成功开发完了吧，这样的事情确实值得我好好庆祝一下

你好！我是你正在访问的这个服务端的开发者。在这个小小的页面里，我确实有点话想留给未来的....各种人。

我似乎并不是一个特别有始有终的人，我有好多项目都曾经半途而废了，这次OurChat服务端的编写也有我努力想挑战一下自己的成分，写一个完成度高的真正生产级别的服务。项目进度很缓慢，几经周折最后还是在这里重新开始了。

这样一件事大概确实不是那么简单，对于身处高二的我也是这样，时间大概不但是从海绵里挤出来的，更是从OI中盗窃出来的。

为什么要开发这样一个项目？为什么要为它创建文档网站，为什么要为它注册域名？为什么要在一个不会有第三个人阅读的网站上提供中英双语？我不知道。我精心地装点它，让它看起来更加郑重，让它看起来更实在，仿佛真的有很多人在使用它，就像孩子天真的给洋娃娃穿上新装，成为它的爸爸或者妈妈。我亲手创造了他的一切然后再一厢情愿地给他穿上一层自以为是的伪装，不得不说，这确实是一件很精致的游戏。

可是就算是游戏我也想把它进行下去，也许我确实能坚持很长一段时间，假如我的运气很好的话，也许会有一个或者两个人站在我的 OurChat 旁边说：“真是一个不错的玩具！”。当然，这样就已经足够了。

还有什么要说的呢，也许我确实应该感谢一下小学生，如果是一个团队的话就没那么容易放弃了吧？大概是这样，不然可能我早就奔向操作系统的开发了，用另一个人的努力确实比较牢固的捆住了我。
